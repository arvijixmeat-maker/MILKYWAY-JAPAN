## 4. 작성 워크플로

1. 1편씩 진행 (병렬 X — 한 편 끝나야 다음)
2. Claude가 H2/H3 구조 + 본문 초안 작성 (8,000~10,000자)
3. 사용자 검토 → 사실관계 (가격, 일정, 현지 정보) 보정
4. **관리자 페이지에서 등록** (코드 작업 아님 — 매거진 카테고리/태그 입력 시 「観光」「準備」「情報」 등 정확히)
5. 등록 후 GSC에서 **인덱싱 요청** (URL 검사 → 색인 생성 요청)

## 6. 주의 사항

- **관리자 페이지에서 카테고리/태그 정확히 입력** — 「관련 ツアー」 자동 매칭이 정확도 ↑
- **상품 tags에 일본어 키워드 보강** (예: 「ゴビ砂漠」 상품에 tags `["ゴビ砂漠", "砂漠", "アドベンチャー"]`) — 매거진 매칭 정밀도 ↑
